{ Beacon }


Introduction


How calculate my daily electricity consumption? How save money by managing monthly electricity consumption. It’s all about consuming electricity in efficient manner. Beacon will solve these all problems. Using Beacon you can,

1. Calculate Electricity Consumption day by day for your tariff category
2. Calculate any Electricity Bill at any time anywhere
3. Analyze your daily and monthly electricity consumption using graphs
4. Contact Ceylon Electricity Board immediately for interruptions of electricity supply

And,The best part is,
You are able to set a Power consumption threshold limit & Bill cycle refresh days to predict how should be your future electricity consumption.


How Beacon Works?


As an example think that you need to limit your number of consumption units as 60 for 30 days. Set these values in Settings and that’s all. Beacon will calculate day by day your daily consumption rate and warn you before exceeding your threshold limit.

We do have colorful beacons for each status of power consumption.

Red Blinking Beacon warn you when your power consumption is critical
Green Beacon tells you that you are consuming electricity in normal rate
Yellow Beacon tells you that you have already activated the bill cycle service

Activate Beacon


If you have activated both services, power consuming rate will be calculated and warn you when the rate is in critical or exceed of units for bill cycle days
If you have activated only the threshold service, beacon will warn when you exceed your threshold unit limit
If you have activated only the bill cycle service, the monthly bill will be refreshed when given number of days exceeds.

Now it’s time to save your money and converse electricity in efficient manner!

Technologies Used by the App


Beacon has developed by using Android Studio which is free and open source development tool for all the android app developers. To store data & information internally The Room persistence library has used. This library provides an abstraction layer over SQLite to allow fluent database access while harnessing the full power of SQLite.
